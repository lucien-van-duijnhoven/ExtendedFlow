Flow.Launcher.Plugin.ExtendedFlow
==================

A plugin for the [Flow launcher](https://github.com/Flow-Launcher/Flow.Launcher).

### Usage

    ef <arguments>


### development

Make a symlink with the build output to the Flow plugin directory:

    mklink /J [C:\path\to\Flow\plugins]\[pluign name] [C:\path\to\build\output\dir]
